📊 Samsung Sales Analysis Dashboard

A complete Supply Chain & Sales Analytics Dashboard built to analyze Samsung’s business performance across multiple dimensions such as Revenue, Inventory, Shipment, Suppliers, and Customers.

This project provides actionable insights into business operations using interactive visualizations.

🚀 Project Overview

The dashboard is designed to help stakeholders understand:

📈 Sales & Revenue trends
🚚 Shipment performance & delays
🏭 Supplier efficiency
📦 Inventory & stock analysis
👥 Customer & channel insights

🛠️ Tools & Technologies Used
Power BI – Data Visualization & Dashboard Creation
Excel / CSV – Data Source & Cleaning
DAX (Data Analysis Expressions) – Measures & Calculations
Data Modeling – Relationships & Schema Design

📌 Key Features
🔹 1. Home Page
Navigation panel for all sections
Clean UI with category-based access
🔹 2. Overview Dashboard
Total Revenue, Profit, Margin KPIs
Supplier, Manufacturer & Shipment summary
Order Quantity & Inventory tracking
Revenue by platform
🔹 3. Supplier Analysis
Supplier performance by:
Lead Time
Order Quantity
Quality Score
Country-wise supplier comparison
🔹 4. Inventory & Production
Inventory value trends
Safety stock & reorder points
Defective units analysis
Turnover rate insights
🔹 5. Shipment Analysis
Shipment delays by carriers
Monthly shipment cost trends
Delivery status distribution
🔹 6. Customer Analysis
Revenue by channel (Online, Retail, Direct)
Profit contribution by platform
Discount analysis
Monthly growth trends

📊 Key Insights
📌 Flipkart & Amazon are top revenue contributors
📌 Certain suppliers have higher lead times impacting delivery
📌 Shipment delays mostly concentrated in specific carriers
📌 Inventory fluctuations affect turnover rates
📌 Discounts directly influence profit margins
